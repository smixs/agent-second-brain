# Cloudflare Email Routing for Windcrest AI

## Setup
- Configure Cloudflare Email Routing for the Windcrest AI domain(s).
- Add the forwarding address or inbox destination that should receive routed mail.
- Verify MX and DNS records before turning routing on.

## Notes
- Keep routing aliases aligned with the operational inbox used by Windcrest AI.
- Test forwarding, aliases, and any catch-all behavior after activation.
